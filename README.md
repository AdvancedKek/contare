# Contare
## Stack:
  - Java 21
  - Spring Boot 3.3
  - PostgreSQL 16
  - SolidJS

## Build project
```shell
./gradlew clean && ./gradlew openApiGenerate && ./gradlew build
```




## Build project to jar
```shell
./gradlew bootJar
```

## Run project
```shell
./gradlew bootRun
```


for Windows:
## Build project
```shell 
(.\gradlew clean) -and (.\gradlew openApiGenerate) -and (.\gradlew build)
```


## Build project to jar
```shell
.\gradlew bootJar 
```

## Run project
```shell
.\gradlew bootRun
```
### API 
http://localhost:8080/api/swagger-ui/index.html
